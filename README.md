# Neovim Config

My Neovim config, based off of [LazyVim](https://github.com/LazyVim/LazyVim).

## Installation

Clone this repo into `~/config/nvim`:

```sh
git clone git@github.com:unindented/neovim-config.git ~/config/nvim
```

Then start Neovim:

```
nvim
```

> **Note**
>
> Run `:checkhealth` to identify and resolve any issues.
